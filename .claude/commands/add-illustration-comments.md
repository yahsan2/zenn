# 差し絵コメント挿入コマンド

記事を読み、差し絵を入れると効果的な箇所にAI画像生成用の指示コメントを挿入する。

## 引数

- $ARGUMENTS: 対象の記事ファイルパス

## 手順

1. 指定された記事ファイルを読み込む
2. 記事の流れを理解し、差し絵が効果的な箇所を特定する（5〜10箇所程度）
3. 各箇所に以下のフォーマットでHTMLコメントを挿入する

## コメントフォーマット

```html
<!--
🎨 差し絵: [シーンのタイトル]
【テイスト】シンプルな線画風、モノクロ基調（黒い線）で水色のアクセントカラー、手描き風、白背景、コミカルでかわいい雰囲気
【アスペクト比】4:3
【注意】画像内に文字・テキストは入れない
【内容】
- [具体的な構図の説明]
- [キャラクターの配置]
- [表情やアクション]
- [エフェクトや背景]
-->
```

## 差し絵を入れるべき箇所の基準

- 新しい概念や登場人物が出てくる場面
- 重要な転換点やクライマックス
- 問題発生や失敗のシーン
- 成功や解決のシーン
- まとめや全体図が必要な場面

## 注意事項

- 画像内に文字・テキスト・セリフは入れない（アイコンや記号で表現）
- 吹き出しのセリフは削除し、表情やジェスチャーで感情を表現
- 看板や標識の文字は警告マークや情報アイコンに置き換え
- 水色をアクセントカラーとして効果的に使う指示を含める
- 具体的で再現可能な指示にする

## 出力

コメント挿入後、イラストレーターへの依頼文を出力する。

### 出力フォーマット

画像出力先は記事ファイル名（拡張子なし）をディレクトリ名として使用する。
例: `articles/claude-code-potemkin-understanding.md` → `images/claude-code-potemkin-understanding/`

```
記事の差し絵を作成してください。

**対象記事:** [記事ファイルパス]
**画像出力先:** images/[記事ファイル名（拡張子なし）]/
**参考画像:** ./images/character01.jpg, ./images/character02.jpg
**差し絵の数:** [挿入したコメントの数]箇所

記事内の `<!--🎨 差し絵: ...-->` コメントに、各画像の指示が書いてあります。
コメントを読んで画像を生成し、上記の出力先に保存してください。
```
